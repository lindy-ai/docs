## Overview

Lindy Meeting Recorder joins your meetings, records them, transcribes conversations, and sends you comprehensive notes with action items - all automatically.

  <img src="/lindy-brand-assets/lindy-meeting-recorder.png" alt="Meeting Recorder workflow showing Calendar trigger, recording, and follow-up actions [[memory:2619471]]" />

## How It Works

### 1. Calendar Event Trigger
- Activates 1 minute before scheduled meetings
- Checks if event contains a meeting link
- Filters for specific meeting types (optional)

### 2. Core Recording Actions
**Record Meeting**
- Joins with customizable message
- Records audio/video and generates real-time transcription
- Appears as "Lindy Meeting Notetaker"

**Get Task URL**
- Captures link to recording and transcript
- Enables sharing in emails and messages

### 3. Smart Email Delivery
Sends comprehensive meeting notes including:
- Action items and deadlines
- Meeting summary
- Direct link to recording/transcript
- Reply-to-email support for follow-up questions

### 4. AI Follow-Up Assistant
**Be Helpful via Chat**
- Notifies you can ask questions about the meeting
- Provides ongoing support through chat

**Answer Follow-up Questions**
- AI agent responds to queries about meeting content
- Clarifies discussed topics
- Helps with action items

## Common Use Cases

- **Sales Calls** - Capture MEDDPICC notes and CRM updates
- **Recruiting** - Document interviews and candidate feedback  
- **Team Meetings** - Track action items and decisions
- **Client Meetings** - Generate professional summaries

<Tip>
Add trigger filters to control which meetings Lindy joins - perfect for excluding internal standups or focusing on external calls.
</Tip>
