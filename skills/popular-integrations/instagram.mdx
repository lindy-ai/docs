Instagram integration enables comprehensive data collection from Instagram profiles, posts, reels, and comments. Perfect for social media monitoring, content analysis, and competitor research.

### Content Collection by URL

  <Accordion title="Instagram reels by URL">

  </Accordion>
  
  <Accordion title="Instagram posts by URL">

  <Accordion title="Instagram comments by URL">

### Advanced Profile Scrapers

<AccordionGroup>
  <Accordion title="Instagram Profile Scraper">
    Scrape all Instagram profile info. Just add one or more Instagram usernames and extract number of followers&follows, URLs, bio, posts, likes, counts, related profiles, captions, highlight reels. Export scraped data, run the scraper via API, schedule and monitor runs or integrate with other tools.
    
    Input: Instagram usernames (one or multiple)
    
    Returns: Comprehensive profile analysis including:
    - Follower and following counts
    - Bio and profile URLs
    - Post counts and engagement metrics
    - Related profiles and connections
    - Highlight reel data
  </Accordion>
  
  <Accordion title="Instagram Post Scraper">
    Scrape Instagram posts. Just add one or more Instagram usernames and get your data in seconds including text, hashtags, mentions, comments, images, URLs, likes, locations, and metadata. Export scraped data, run the scraper via API, schedule and monitor runs or integrate with other tools.
    
    Input: Instagram usernames
    
    Returns: Detailed post analysis including:
    - Text content and hashtags
    - Mentions and comments
    - Images and media URLs
    - Location data and metadata
    - Engagement metrics
  </Accordion>
  
  <Accordion title="Instagram Comments Scraper">
    Scrape Instagram comments from posts or reels. Just add one or more Instagram post URLs to get comment text, post IDs, position, timestamp, owner ids, usernames, profile pics, and URLs. Export scraped datasets, run the scraper via API, schedule and monitor runs or integrate with other tools.
    
    Input: Instagram post URLs
    
    Returns: Comprehensive comment data including:
    - Comment text and timestamps
    - User information and profile pictures
    - Post IDs and positioning data
    - Owner details and URLs
  </Accordion>
</AccordionGroup>
