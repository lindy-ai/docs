## Quick Overview

Lindy will:

- **Trigger** whenever a résumé arrives by email, folder upload, or ATS webhook  
- **Parse** the document into structured data (name, skills, experience)  
- **Filter** out obvious mismatches (years, must-have skills)  
- **Score** remaining candidates with an AI agent and two-line rationale  
- **Route** A-players to Slack and log everyone else in Google Sheets  

Build the flow once—then let Lindy pre-screen every application 24/7.

## Setting Up Your Resume Screener

### Step 1: Capture Incoming Résumés

<Frame>
  <img src="/lindy-brand-assets/placeholder.png" alt="Resume trigger" />
</Frame>

1. Use one or more triggers:  
   • `Gmail – Email Received` (filter subject contains "Application")  
   • `Google Drive – File Added` in folder *Resumes/2025*  

2. OPTIONAL: add an ATS webhook when a new applicant is created.

### Step 2: Parse the Document

<Frame>
  <img src="/lindy-brand-assets/placeholder.png" alt="Parse resume" />
</Frame>

- Action: `File – Extract Text` (handles PDF, DOCX, TXT).  
- Store output as `resumeText`.

### Step 3: Extract Key Fields

<Frame>
  <img src="/lindy-brand-assets/placeholder.png" alt="AI extract fields" />
</Frame>

- Action: `AI Agent – Extract JSON`  
- Prompt:

```
From resumeText, return JSON:
name, email, years_experience, skills[], last_company, education
```

### Step 4: Basic Filters

<Frame>
  <img src="/lindy-brand-assets/placeholder.png" alt="Filter conditions" />
</Frame>

- **Condition 1**: years_experience < 2 → stop flow.  
- **Condition 2**: skills missing "React" OR "TypeScript" → stop flow.

### Step 5: Score & Summarize

<Frame>
  <img src="/lindy-brand-assets/placeholder.png" alt="AI score" />
</Frame>

- Action: `AI Agent – Generate Text`  
- Prompt:

```
You are a Tech Recruiter.

Using extractedJSON, rate fit for "Frontend Engineer" 1-5 and give a 2-sentence justification.
Return:
"score": X, "summary": "…"
```

### Step 6: Route the Outcomes

| Path | Action |
|------|--------|
| score ≥ 4 | `Slack – Send DM` to #hiring with summary + résumé attached |
| score 2-3 | `Google Sheets – Append Row` to tab *Warm Pipeline* |
| score ≤ 1 | (Optional) `Gmail – Send Reply` polite rejection |

### Step 7: (Optional) LinkedIn Enrichment

<Frame>
  <img src="/lindy-brand-assets/placeholder.png" alt="LinkedIn search" />
</Frame>

- `Search the Internet` → "site:linkedin.com/in name last_company"  
- Append profile URL to Slack or sheet entries.

<Tip>
If you need exact statistics (median score, number passed) for weekly hiring stand-ups, insert a **Run Code – Python** step before the Slack action to calculate and include those numbers in the DM.
</Tip>

## Advanced Customizations

| Trick             | What it does                                                      |
|-------------------|--------------------------------------------------------------------|
| Bias-Blind Scoring | Strip name & school in extract step to create anonymized reviews. |
| Keyword Boost      | +1 score if skills include "GraphQL" or "Performance Tuning".     |
| Attach PDF         | Slack DM can include the original résumé file for quick skim.     |
| Auto-Calendar      | High scorers → trigger Meeting Scheduler to book intro screen.    |

## Best Practices

<AccordionGroup>
  <Accordion title="Standardize Skills">
    Keep a canonical skill list so "ReactJS" and "React.js" map to one tag.
  </Accordion>
  
  <Accordion title="Human Calibration">
    Review Lindy's first 20 scored résumés to fine-tune thresholds.
  </Accordion>
  
  <Accordion title="Privacy Compliance">
    Store résumés only as long as policy allows; auto-purge after 6 months.
  </Accordion>
  
  <Accordion title="Feedback Loop">
    Add a column for human hire/no-hire outcome to retrain future prompts.
  </Accordion>
</AccordionGroup>
